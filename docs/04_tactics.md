# 04. 戦術セット仕様

## 目的

本ドキュメントは、戦術プリセット、戦術スロット編集、自然言語戦術生成に関する仕様を定義する。

## 基本方針

バトルはオートで進行する。  
プレイヤーは戦闘前に、各機体へ戦術セットをインストールする。

戦術セットは、機体が戦闘中にどの行動を優先するかを定義する。

```text
機体 × 戦術セット
```

## 初心者向けプリセット

初心者は、完成済みの戦術プリセットを選択する。

初期プリセットは以下の5種類とする。

1. 近接戦闘型
2. 中距離ヒットアンドアウェイ型
3. 遠距離狙撃型
4. 爆撃型
5. 砲台型

## 近接戦闘型

短距離で高火力を出す前衛向け戦術。

```text
自分HPが低い → 防御
相手が遠距離型 → 接近攻撃
ENが十分 → 高威力近接
相手HPが低い → 追撃
それ以外 → 通常攻撃
```

向く機体：

- 獣型
- 高HP
- 高DEF
- 高ATK

## 中距離ヒットアンドアウェイ型

回避と削りを重視する戦術。

```text
相手が近接型 → 回避攻撃
自分HPが低い → 距離を取る
ENが十分 → 中威力攻撃
相手HPが低い → 追撃
それ以外 → 通常攻撃
```

向く機体：

- 鳥形
- SPD高め
- TEC高め

## 遠距離狙撃型

高命中・高火力の単体攻撃を重視する戦術。

```text
相手HPが低い → 狙撃
相手が鳥形 → 命中重視攻撃
ENが不足 → チャージ
ENが十分 → 高威力狙撃
それ以外 → 通常射撃
```

向く機体：

- 人型
- 鳥形
- TEC高め

## 爆撃型

範囲攻撃と複数敵処理を重視する戦術。

```text
敵が複数残っている → 範囲攻撃
相手が防御中 → 爆撃
ENが不足 → チャージ
相手後衛が残っている → 後衛攻撃
それ以外 → 通常攻撃
```

向く機体：

- 人型
- 獣型
- EN高め

## 砲台型

固定火力、防御、迎撃を重視する戦術。

```text
相手が接近型 → 迎撃
自分HPが高い → 砲撃
自分HPが低い → 防御
ENが十分 → 重砲撃
それ以外 → 通常砲撃
```

向く機体：

- 獣型
- 人型
- DEF高め

## 中級者向けスロット編集

中級者は、プリセット戦術の中身を編集できる。

標準仕様は以下とする。

```text
4スロット + 基本行動
```

例：

```text
スロット1：自分HPが30%以下 → 防御
スロット2：相手が鳥形 → 命中重視攻撃
スロット3：自分ENが80以上 → 高威力攻撃
スロット4：相手HPが25%以下 → 追撃
基本行動：通常攻撃
```

## 判定方式

```text
1. スロット1から順に条件を評価
2. 条件を満たした最初の行動を選択
3. 行動不能なら次のスロットを評価
4. すべて不成立なら基本行動を実行
```

## 条件候補

条件は全プレイヤー共通とする。  
課金によって増えない。

| 分類 | 条件例 |
|---|---|
| 自分条件 | HPが一定以下、ENが一定以上、状態異常中、前ターン行動 |
| 相手条件 | 型、HPが一定以下、DEFが高い、SPDが高い、前ターン行動 |
| 戦況条件 | ターン数、敵残数、味方残数、自分の位置、敵の位置 |

## 行動候補

行動は全プレイヤー共通とする。  
課金によって増えない。

| 行動タグ | 内容 |
|---|---|
| 通常攻撃 | 安定した攻撃 |
| 高威力攻撃 | 火力重視 |
| 命中重視攻撃 | 高速・回避型に有効 |
| 貫通攻撃 | 高DEF相手に有効 |
| 範囲攻撃 | 複数敵に有効 |
| 防御 | 被ダメージ軽減 |
| 回避行動 | 被弾率低下 |
| チャージ | EN増加 |
| 妨害 | 相手能力を下げる |
| 追撃 | HPが低い相手を倒し切る |
| 接近攻撃 | 遠距離型に詰める |
| 迎撃 | 接近相手に有効 |
| 後衛攻撃 | 後衛を狙う |

## 自然言語戦術生成

課金ユーザーは、文章から戦術スロットを生成できる。

ただし、これは入力補助であり、戦闘性能を直接高めない。

原則：

1. 自然言語は通常の戦術スロットへ変換する。
2. 戦闘中にLLMは使わない。
3. 課金しても条件・行動・スロット数は増えない。
4. 生成された戦術は非課金ユーザーも手動で再現可能。

## 自然言語生成の流れ

```text
1. 作戦文を入力
2. 3〜5スロットへ変換
3. 変換結果を表示
4. プレイヤーが確認・修正
5. 保存
```

## 例

入力：

```text
鳥形相手には命中重視で、HPが減ったら防御。
ENが溜まったら必殺技で倒し切る。
```

変換結果：

```text
スロット1：自分HPが30%以下 → 防御
スロット2：相手が鳥形 → 命中重視攻撃
スロット3：自分ENが80以上 → 高威力攻撃
スロット4：相手HPが25%以下 → 追撃
基本行動：通常攻撃
```
