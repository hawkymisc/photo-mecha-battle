# 03. メカ生成・型・ステータス仕様

## 目的

本ドキュメントは、抽出オブジェクトからのメカ生成、メカ型、情報量スコア、ステータス決定に関する仕様を定義する。

## メカ生成の入力

- 元画像
- 抽出オブジェクト画像
- セグメンテーションマスク
- 輪郭画像
- エッジ画像
- 画像特徴量
- 選択されたメカ型

## メカ型

メカは以下の3型とする。

| 型 | 役割 |
|---|---|
| 鳥形 | 高速、回避、先制 |
| 人型 | バランス、武装切替、戦術適性 |
| 獣型 | 装甲、近接、耐久 |

## 鳥形メカ

高速・回避・先制に優れる。

| 項目 | 内容 |
|---|---|
| 得意 | 回避、先制、空中攻撃 |
| 苦手 | 高装甲、長期戦 |
| 向く素材 | 細長い物、軽そうな物、枝、葉、傘、工具 |
| ステータス傾向 | SPD高、EVA高、HP低め |

## 人型メカ

バランス・武装切替・命中に優れる。

| 項目 | 内容 |
|---|---|
| 得意 | 汎用性、スキル連携、命中 |
| 苦手 | 極端な特化型との相性差 |
| 向く素材 | 道具、家電、文房具、機械部品、人工物 |
| ステータス傾向 | 平均的、TEC高め |

## 獣型メカ

耐久・近接・突進に優れる。

| 項目 | 内容 |
|---|---|
| 得意 | HP、DEF、近接火力 |
| 苦手 | 高速回避、空中戦、妨害 |
| 向く素材 | 靴、バッグ、石、ぬいぐるみ、動物、塊感のある物 |
| ステータス傾向 | HP高、DEF高、SPD低め |

## 画像特徴量

抽出オブジェクトから以下を計算する。

| 分類 | 内容 |
|---|---|
| 情報量 | 輝度エントロピー、色分布エントロピー |
| 色 | 主要色、色数、彩度、明度 |
| 輪郭 | 輪郭長、凸包比、穴、シルエット複雑度 |
| 形状 | 面積、縦横比、丸さ、細長さ、対称性 |
| 質感 | エッジ密度、高周波成分、パターン性 |
| 品質 | ブレ、暗さ、露出、ピント |
| 意味情報 | 推定カテゴリ、希少性、人工物/自然物傾向 |

## 情報量スコア

```text
ObjectInfoScore =
  0.25 * VisualEntropy
+ 0.20 * EdgeComplexity
+ 0.15 * ColorDiversity
+ 0.15 * ShapeComplexity
+ 0.10 * SemanticRarity
+ 0.10 * CaptureQuality
+ 0.05 * SizeBalance
```

情報量スコアは、強さの一部として扱う。  
単純に総合戦闘力へ直結させない。

## 基本ステータス

| ステータス | 内容 |
|---|---|
| HP | 耐久力 |
| ATK | 攻撃力 |
| DEF | 防御力 |
| SPD | 行動順、回避、先制 |
| TEC | 命中、スキル発動、戦術安定性 |
| EN | 特殊行動リソース |
| LUCK | 低確率効果、クリティカル補正 |

## 画像特徴とステータスの対応

| 画像特徴 | 反映先 |
|---|---|
| 高エントロピー | EN、特殊攻撃、レアリティ |
| エッジ密度が高い | 武装数、命中、クリティカル |
| 色数が多い | 属性スロット、状態異常耐性 |
| 面積が大きい | HP、DEF |
| 細長い形 | SPD、射程 |
| 丸い・塊感がある | DEF、突進、耐久 |
| 左右対称 | TEC、命中、制御性能 |
| 希少カテゴリ | 固有スキル、称号、特殊適性 |

## 画像生成方針

- 元オブジェクトの色やシルエットをある程度残す
- 鳥形・人型・獣型の識別性を高める
- 既存IP風の出力を避ける
- 実在兵器に過度に寄せない
- 過度な暴力表現を避ける

## 重要原則

生成画像は見た目を表す。  
戦闘性能は、生成画像ではなく、撮影画像と抽出オブジェクトの分析値から決定する。
